	6.http://localhost:8001/index.html

## 使用步骤
   1.下载项目到本地目录。
   2.找到项目config/default.json 配置文件，修改kafak数据库连接地址。
   3.执行app.js文件
   4.执行test.js文件
   5.打开浏览器输入 http://localhost:8001/index.html。
